# Irresistible

A simple script to find a resistance based on a list of resistors given. 

The script only goes 2 nested layers deep. This means the maximum number of resistors per resistance value you query will be 4. 


For example:
- (R1||R2)||(R3||R4) 

- (R1+R2+R3+R4)
